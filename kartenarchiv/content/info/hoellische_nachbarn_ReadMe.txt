I. Einleitung
=========
Höllische Nachbarn ist eine kleine Karte mit allem, was man braucht: Gold, Ressourcen, Kreaturengeneratoren. Es gibt zwei Spieler (Rot und Blau), die jeweils eine Hälfte der Karte ihr eigen nennen. Als Abgrenzung dient ein Gebirge, das zwei Durchgänge besitzt, die jeweils von 2 Grenzwachen blockiert werden. Außerdem gibt es noch einen unterirdischen Tunnel, der auf gleiche Weise abgesperrt ist.
